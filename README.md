# vuejs-crash-course
Creating a Vue JS app with JSON server API backend

## Project setup
```
npm install
```

### Compiles and hot-reloads for development (serve for front end, dev for JSON server)
```
npm run serve
npm run dev
```
Local server
http://localhost:8080/

API Route Example
http://localhost:5000/tasks/1

### Compiles and minifies for production
```
npm run build
```
